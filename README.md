# Electron-vue3

本模板采用[electron-vue-vite](https://github.com/caoxiemeihao/electron-vue-vite)为雏形搭建。

客户端打包使用[electron-builder](https://www.electron.build/)，相关配置在`/electron-builder.json`中配置。
